## Description

Ce projet est une application en Python simulant un tirage de loto.

Le programme génère 10 numéros aléatoires compris entre 1 et 49. L’utilisateur saisit ses propres numéros, puis le programme compare les résultats et attribue une récompense en fonction du nombre de correspondances.

---

## Fonctionnalités

- Tirage aléatoire de 10 numéros sans doublons
- Saisie utilisateur avec vérifications
- Comparaison des numéros
- Comptage des numéros corrects
- Attribution d’une récompense
- Bonus : vérification des numéros bien placés
